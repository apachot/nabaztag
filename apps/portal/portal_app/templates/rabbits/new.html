Ajouter un lapin

Commence par créer la fiche du lapin ici, puis suis la procédure de configuration historique du Nabaztag. Le but est de lui faire rejoindre ton Wi-Fi et de lui indiquer l’adresse de notre serveur.

Important : sur un Nabaztag d’origine, la configuration se fait depuis le Wi-Fi émis par le lapin, puis via son interface locale sur http://192.168.0.1/.

1. Créer la fiche du lapin

Optionnel au départ. Tu pourras le renseigner après le provisioning si besoin.
Retour

2. Manipulation sur le lapin

  1. Débranche le Nabaztag.
  2. Maintiens le bouton appuyé pendant que tu le rebranches.
  3. Attends qu’il passe en mode configuration, généralement avec une lumière bleue.
  4. Depuis ton ordinateur, cherche le réseau Wi-Fi émis par le lapin, par exemple Nabaztag8E.
  5. Connecte-toi à ce réseau Wi-Fi temporaire.
  6. Ouvre ensuite http://192.168.0.1/ dans ton navigateur.
  7. Va dans Advanced configuration ou l’écran de configuration avancée équivalent.

3. Valeurs à saisir dans l’interface web du lapin

Les libellés exacts peuvent varier légèrement selon la version du lapin, mais on retrouve en général SSID, la clé Wi-Fi, puis Violet Platform.

Si l’écran du lapin affiche un bouton comme Update and Start, c’est celui qu’il faut utiliser une fois les champs remplis.

4. Après validation

  1. Le lapin redémarre et tente de rejoindre ton Wi-Fi maison.
  2. Reviens sur ce portail et ouvre la fiche du lapin.
  3. Utilise ensuite la page Provisioning ou les actions de connexion pour finaliser les tests.
